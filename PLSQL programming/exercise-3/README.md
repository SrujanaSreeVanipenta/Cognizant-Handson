# Exercise 3: Stored Procedures

## Objective

To implement PL/SQL stored procedures for common banking operations.

## Scenario 1: Process Monthly Interest

Implemented a procedure `ProcessMonthlyInterest` to apply 1% interest to all savings accounts.

## Scenario 2: Update Employee Bonus

Implemented a procedure `UpdateEmployeeBonus` that updates employee salaries based on department and bonus percentage.

## Scenario 3: Transfer Funds

Implemented a procedure `TransferFunds` that transfers money between accounts after checking sufficient balance.

## Outcome

Successfully created, compiled, and executed all procedures in Oracle SQL Developer and verified the results using SQL queries.
